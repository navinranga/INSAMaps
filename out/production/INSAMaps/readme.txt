CONTENU DE L'ARCHIVE "INSAMaps":

- Dossier ".idea" (contenant les fichiers de paramètres spécifiques au projet d'IntelliJ IDEA donc à ignorer)
- Dossier "autres" (contenant 
	fichier excel "catalogue.xlsm" indispensable pour le programme
	fichier excel "database.xlsx" à consulter pour découvrir notre base de donnée,
	fichier png "uml.png" diagramme UML du projet,
	fichier svg "uml.svg" diagramme UML à consulter à l'aide d'un lecteur de fichier svg comme firefox pour un meilleur aperçu,
	fichier publisher "wallpaper.pub" l'arrière plan de l'animation utilisée dans le programme,
	fichier txt "distance.txt" contenant matDistance de notre programme,
	fichier txt "brouillon1" donc à ignorer,
	fichier txt "brouillon2" donc à ignorer,
	fichier pdf "gr98_cahier_charges_rang_smek_tran_vieg.pdf" notre cahier de charges,
	fichier pub "listes v0 bat et rues.pub" contenant les premiers essais de réduction de problème
	)
- Dossier "out" (contenant l'output du projet quand on compile avec IntelliJ IDEA donc à ignorer)
- Dossier "pictures" (contenant l'ensemble des images utilisées dans notre projet)
- Dossier "rapport" (contenant le rapport "rapport_vf.docx" et "rapport_vf.pdf)

- Fichier java "AnimationDeb.java" (class java utilisée au début du programme comme animation de début)
- Fichier java "Batiment.java" (class java définissant un Batiment avec ses caractéristiques)
- Fichier java "Dijkstra.java" (class java permettant implémentation de l'algorithme de Dijkstra et récupération des caractéristiques de notre trajet)
- Fichier java "FenetreCarte.java" (class java qui contient la carte, la partie dédiée à l'IHM)  
- Fichier java "FenetreGraphique.java" (class java permettant d'avoir la carte comme fond de notre fenêtre et afficher le chemin)
- Fichier java "Graphe.java" (class java permettant de créer un graphe avec la matrice d'entrée)
- Fichier java "Main.java" (class java à exécuter pour faire fonctionner notre programme)

- Fichier iml "INSAMaps.iml" (INSAMaps.iml utilisé par InjtelliJ IDEA pour sauvegarder la configuration de module)
- Fichier txt "readme.txt" (fichier à consulter avant de consulter le projet)

*******************************

EXECUTION DU PROJET
- Ouvrir votre terminal
- Accéder au répertoire dans lequel le programme est enregistré
- Écrire la commande 'javac Main.java' et entrer
- Écrire la commande 'java Main.java' et entrer
